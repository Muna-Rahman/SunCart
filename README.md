# SunCart 

SunCart is a modern and responsive web application built with Next.js. It allows users to browse products, create accounts, sign in securely, and manage their profiles through a clean and user-friendly interface.

🔗 Live Site: https://sun-cart-one.vercel.app/

---

## Purpose

The goal of SunCart is to practice and demonstrate modern full-stack web development using Next.js, MongoDB, and Better Auth. The project focuses on authentication, protected routes, responsive UI design, and session management in a real-world style application.

---

## Features

- User registration and login system
- Secure email and password authentication
- Google OAuth sign-in support
- Protected routes for authenticated users
- Responsive product showcase section
- User profile dashboard
- Persistent login sessions
- Password show/hide toggle
- Smooth animations and modern UI design
- Mobile-friendly responsive layout
